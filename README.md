# Supermarket Sales Dashboard (Power BI)

Interactive Power BI dashboard analyzing supermarket sales performance.

## Tech Stack
- Power BI
- DAX
- Power Query

## Key Features
- Revenue, Net Revenue, Profit, and GPM (%)
- Sales and profit trend analysis
- Product line and payment method comparison
- Interactive filters by customer type and time
